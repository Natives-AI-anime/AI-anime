# 파이썬

## 파이썬 프로젝트 만들기

일단 파이썬을 다운로드를 했다는 가정 하에 우리는 파이썬을 실행할 수 있는 가상 환경에서 파이썬 프로젝트를 생성 및 이용할 수 있음. 이는 파이썬 프로젝트들의 충돌을 방지하기 위함. 

각 프로젝트에서 사용하는 여러 외부라이브러리들은 서로 다르고 같더라도 버전이 다를 수도 있기 때문에 로컬의 전역에 라이브러리를 설치하는 것이 아닌 가상환경에 설치하여 충돌 방지를 함.

이걸 위해서 

```powershell
$ python -m venv .venv
```

라는 명령어를 이용하여 가상 환경을 생성한다.

이후 

``` powershell
$ .venv\Scripts\activate
```

명령어를 통해서 가상 환경에 접속한다.

그리고 이곳에서 프로젝트 작업을 하면 되는 것이다.

그리고 파이썬을 이용하는 가장 큰 이유 중 하나인 외부 라이브러리 설치는 pip를 이용해서 한다.

``` powershell
(.venv) $ pip install fastapi uvicorn "langchain>=0.2.0" google-genai
```

pip install 뒤에 설치하고자 하는 라이브러리 이름을 적어주면 된다.

이후

``` powershell
(.venv) $ pip freeze > requirements.txt
```

명령어를 통해서 requirements.txt에 각 라이브러리의 버전을 기록하면 된다.



이후에 할 것은 프로젝트 구조를 만드는 것인데

나의 경우에는 메인 기능을 구현하는 app이라는 패키지와 여러 정적 데이터나 각 개발자의 환경에 따라 다를 환경 변수들을 저장하고 여러 기본 설정값들을 정의하는 config 패키지를 구성하여 작업을 하도록 할 것이다.

이 때, 우리는 각 디렉터리에

\_\_init\_\_.py 파일을 생성해주어 패키지로 인식을 하게 된다.

그리고 .gitignore 파일을 만들어 루트 디렉터리에 추가하여 아까 말한 config 패키지에 들어가는 여러 보안 요소들과 환경 변수 파일 등을 git에 업로드하지 않게 하면 마무리가 된다.
